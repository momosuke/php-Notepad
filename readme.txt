【各バージョン更新履歴】
（version 1）
　　ファイル、フォルダ、機能の３段表示(全画面)
　　ページタイトルなし、文字化けあり
（version 1.1）
　　ﾍﾟｰｼﾞﾀｲﾄﾙ, 文字化け修正、正規表現の改良
（version 2）
　　画面周辺に余白を設置、左右の２段表示(一覧)    
　　処理結果表示については,余白は未適用
　　閲覧ページにフォルダのリンクを作成
（version 3）
　　一部機能の表示位置を変更
　　処理結果表示については,一覧表示に移植
（version 4.0）
　　ログイン認証を設定 (ただし,単一ユーザ及びパスワードのみ)
　　規制ユーザ作成時、ログインする誤作動あり
（version 4.1）
　　ユーザ別にファイルを所有 (共有パスワードを使用)
　　ログイン認証時の誤作動の修正
　　バックアップを削除時に削除し, 表示前に削除を行う
（version 4.2）
　　同一ユーザの規制(共有ﾊﾟｽﾜｰﾄﾞ)
　　新規ユーザ作成時にパスワード認証
（version 5.0）
　　一括ダウンロード機能追加
（version 5.1）
　　ダウンロードファイルをユーザIDに変更
　　編集時にタイトル名を固定,ファイルの再構成(文字化け防止)
　　ログイン機能の整理および修正
（version 6.0）
　　ユーザ対応パスワードの設置
（version 6.1）
　　同一IPアドレスのユーザ規制
　　タイトル変更時のバグ修正
　　存在しないフォルダ番号に対応
　　共有ユーザの設置(一般・滝本研)
